#### Macie job run and findings

1. Select **Show results**, then select **Show CloudWatch logs**.

![Macie](/images/2/00020.png?featherlight=false&width=90pc)

2. Monitor **Log events**

![Macie](/images/2/00021.png?featherlight=false&width=90pc)

3. Select **Show results**

![Macie](/images/2/00022.png?featherlight=false&width=90pc)

4. Select **Show findings**

![Macie](/images/2/00031.png?featherlight=false&width=90pc)

5. In the **Findings** interface

![Macie](/images/2/00032.png?featherlight=false&width=90pc)

6. Select **Action** and select **Export(JSON)**

![Macie](/images/2/00035.png?featherlight=false&width=90pc)

7. Select **Download**

![Macie](/images/2/00033.png?featherlight=false&width=90pc)

8. After about 10-15 minutes you view **Log events** and you will see the job completed.

![Macie](/images/2/00034.png?featherlight=false&width=90pc)

9. Back to the **Amazon Macie** interface, we see the **Summary** section of the **S3 bucket**.

![Macie](/images/2/00036.png?featherlight=false&width=90pc)

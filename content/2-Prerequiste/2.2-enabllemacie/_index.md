#### Enable Macie

1. Access the **Macie** interface

- Select **Get started**

![Macie](/images/1/0007.png?featherlight=false&width=90pc)

2. In the **Get started** interface, select **Enable Macie**

![Macie](/images/1/0008.png?featherlight=false&width=90pc)

3. Activate Macie successfully.

![Macie](/images/1/0009.png?featherlight=false&width=90pc)

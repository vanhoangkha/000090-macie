#### Create a Macie job

1. Return to **Job** configuration interface, select **EmpID** and select **Next**.

![Macie](/images/2/00014.png?featherlight=false&width=90pc)

2. For **Select allow lists** you can skip and select **Next**.

![Macie](/images/2/00015.png?featherlight=false&width=90pc)

3. Do **Enter general settings**, enter **Job name** as **```EmpID```**, then select **Next**.

![Macie](/images/2/00016.png?featherlight=false&width=90pc)

4. Execute **Review and create**

![Macie](/images/2/00017.png?featherlight=false&width=90pc)

5. Select **Submit**

![Macie](/images/2/00018.png?featherlight=false&width=90pc)

6. When newly created **Job** will have status **Active(Running)**

![Macie](/images/2/00019.png?featherlight=false&width=90pc)
